## feup-asso

----

### Instructions

#### Setup
Install pony compiler `ponyc`

#### Compile
```shell
# compile the package
$ make build/asso 

# compile tests
$ make build/test 
```

#### Run

__Note:__ both these commands will compile the source if they're not built.
```shell

# run the package's Main
$ make run

# run tests
$ make test
```
